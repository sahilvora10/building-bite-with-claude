# v0.3 Bugs Found and Fixes

**Date:** 2026-04-05 / 2026-04-06

## Bug 1: Rings not nested (Apple Fitness style)

**Symptom:** Rings were displayed side by side as individual circles, not nested concentrically like Apple Fitness.

**Fix:** Rewrote `ProgressRing.tsx` as `NestedRings` — renders concentric circles with decreasing radius per ring. Outer = calories, inner = fat. Added legend below with colored dots.

**Lesson:** Be specific with visual references in the spec. "Apple-style rings" could mean many things — nested concentric was the intent.

---

## Bug 2: Ring overflow past 100% not visible

**Symptom:** When a ring hit 100%, it looked identical to 99%. No visual indication that the goal was exceeded.

**Fix:** Added a second semi-transparent lap overlaid on the full ring when progress > 1.0. The overflow ring is slightly thinner and 60% opacity, creating a visible "second round" effect.

**Lesson:** Edge cases in visual components matter — design for the states beyond the happy path.

---

## Bug 3: Meal card flickering (persistent)

**Symptom:** Every time a new meal was logged, all existing meal cards visually flickered/re-rendered. Tried multiple fixes:
1. `memo` on MealCard — didn't help
2. `useCallback` on renderMeal — didn't help
3. FlatList with `getItemLayout`, `removeClippedSubviews` — didn't help
4. Replace FlatList with ScrollView — didn't help
5. Extract `MealList` as memo component with custom comparator — testing

**Root cause:** The parent `HomeScreen` re-renders when `totals` or `goals` state changes (which happens after every meal log via `refreshData`). This cascades to all children including the meal list.

**Current fix attempt:** `MealList` is now a separate `memo` component with a custom comparator that checks meal IDs, not object references. This should prevent re-renders when only totals/goals change.

**Lesson:** React re-rendering cascades are the #1 cause of visual jank in React Native. Isolating frequently-changing state from stable UI via component boundaries + memo is the standard fix.
