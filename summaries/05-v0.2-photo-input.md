## How it works
1. User taps "Camera" or "Gallery" button
2. Permission is requested (once)
3. Photo is picked/taken at 0.7 quality with base64 encoding
4. Preview shown with ✕ remove button
5. User can optionally add text context (e.g., "this is palak paneer")
6. Hit Log → photo + optional text sent to Gemini → nutrition result saved
