# Session 12: v0.9 Build — Extended history + meal time tags

**Date:** 2026-04-08

## What was done
- Extended summaries beyond 7 days with "Load older days" button (adds 14 days per tap)
- Added meal time tags derived from logged time: Breakfast (<11am), Lunch (11am-3pm), Snack (3-5pm), Dinner (>5pm)
- Color-coded tags on meal cards in HomeScreen and expanded summaries
- Created shared mealTag utility for tag derivation and colors

## Files created/changed
- `src/utils/mealTag.ts` — new shared utility for meal tag derivation and colors
- `src/screens/SummariesScreen.tsx` — load more button, meal tags in expanded view
- `src/screens/HomeScreen.tsx` — meal tags on today's meal cards
- `src/db/database.ts` — getDailySummaries now accepts configurable days param

## Design decisions
- Meal tags auto-derived from time (no user input needed) — reduces logging friction
- Tag colors: orange (Breakfast), green (Lunch), purple (Snack), blue (Dinner)
- "Load older days" only shows when there's actually more data to load
- Tags shown as small colored pills below meal name
